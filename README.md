# Student Year Level and Grade Program

This program is designed to:

1. Display the **year level** of a student based on numeric input.
2. Display the **letter equivalency** of the student's grade.

---

## Features

### Year Level Display
The program accepts a numeric input to determine the student’s year level:

| Input | Year Level   |
|-------|-------------|
| 1     | Freshman    |
| 2     | Sophomore   |
| 3     | Junior      |
| 4     | Senior      |
| Other | Invalid     |

### Grade Letter Equivalency
The program converts numeric grades into letter grades:

| Numeric Range | Letter Grade |
|---------------|-------------|
| 100 – 96      | A           |
| 95 – 90       | B+          |
| 89 – 85       | B           |
| 84 – 80       | C+          |
| 79 – 75       | C           |
| 74            | INC         |
| < 74          | Failed      |

---

## How to Use

1. Run the program.
2. Enter the student’s year level (1-4).
3. Enter the student’s grade (0-100).
4. The program will display the corresponding year level and grade letter.

---

## Example

Enter year level: 2
Year Level: Sophomore

Enter grade: 91
Letter Grade: B+

## Notes

- Any year level outside 1-4 will be considered invalid.
- Grades below 74 are considered failed.
- Grade 74 is marked as "INC" (Incomplete).
